# Enterprise CRM System

A full-stack CRM system built with Spring Boot, React, MySQL, and Docker.

## Features

- JWT authentication
- Role-based access control
- User management
- Customer management
- Lead management
- Contact management
- Task management
- Notes
- Audit logs
- Dashboard analytics
- Basic reports
- Dockerized frontend, backend, and database
- Health checks
- Backup and restore documentation

## Tech Stack

### Backend

- Java 21
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- Maven
- MySQL
- Flyway

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS
- Axios

### DevOps

- Docker
- Docker Compose
- Spring Boot Actuator
- Git
